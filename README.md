# NLP Tutorials

A collection of hands-on Jupyter notebooks for learning **Natural Language Processing (NLP)** step by step, starting from **text pre-processing** and gradually moving towards more advanced topics.

This repository is meant to be a practical companion while you learn NLP concepts, experiment with real code, and build intuition using simple, clear examples.

- Build a solid intuition for core NLP concepts.
- Learn **how** and **why** each pre-processing step is used.

- Serve as a base to extend into more advanced NLP workflows.

### `Pre-processing/` (Current focus)

This folder contains notebooks that typically cover topics like:

* Loading and inspecting raw text
* Lowercasing and basic text cleaning
* Removing punctuation, numbers, and special characters
* Tokenization (splitting text into words/tokens)
* Stopword removal
* Stemming and Lemmatization
* Basic vectorization ideas (like Bag-of-Words, etc., if included)

## üöÄ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Asphane/NLP-Tutorials.git
cd NLP-Tutorials
```

### 2. (Optional) Create a virtual environment

**On macOS / Linux:**

```bash
python3 -m venv venv
source venv/bin/activate
```

**On Windows (PowerShell):**

```bash
python -m venv venv
venv\Scripts\Activate
```

### 3. Install dependencies

Install the main libraries commonly used for NLP experiments:

```bash
pip install jupyter notebook numpy pandas nltk spacy scikit-learn matplotlib
```

(You can add or remove libraries based on what each notebook uses.)

### 4. Launch Jupyter Notebook

```bash
jupyter notebook
```

Then, in the Jupyter interface:

1. Navigate to the `Pre-processing/` folder.
2. Open any `.ipynb` notebook.
3. Run the cells one by one and experiment with the code.
